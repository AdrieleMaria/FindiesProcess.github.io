Task: Questionar
Na tarefa de questionamento deve ser usado uma técnica de entrevista como o “O que? Como? Por que?”
Purpose
Com essa entrevista pode-se, na maior parte das vezes, obter uma resposta mais concreta para ajudar no levantamento do requisito.
Relationships
RolesPrimary:
  • equipe
  • usuario
Additional: Assisting:
Main Description
Essa técnica consiste em utilização de perguntas chave, não somente 3 perguntas, mas várias perguntas nas quais possa entender o que é feito, como é feito e porque é feito.
Steps
perguntas

Um exemplo de utilização dessa técnica é quando se pergunta quando a realização de uma tarefa:


● O que é realizado nessa essa tarefa?
● Como é realizado nessa tarefa?
● Por que precisa ser dessa forma?
