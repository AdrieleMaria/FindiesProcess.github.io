Task: Entender os usuarios
A ferramenta busca caracterizar as pessoas.
Purpose

Para poder entender melhor o tipo de usuário que irá acessar o sistema é indicado usar a ferramenta Personas.

Relationships
RolesPrimary Performer:
  • equipe
  • usuario
Additional Performers:
Main Description
Identificar grupos por características padrões como por exemplo, faixa etária, classe social ou comportamento. É preciso ilustrar o personagem, criar uma história do personagem baseado nas suas características. Isso ajuda a entender para quem estamos desenvolvendo o produto.
Steps
Refinamento dos usuários

Para se obter um perfil dos usuários é preciso separar os seguintes pontos:


● Faixa etária,
● Rotina,
● Perfil comportamental,
● Classe social,
● entre outras a ser definidas de acordo com os usuários.
